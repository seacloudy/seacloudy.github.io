Android Studio Support Library

最新的Android版本通常会为你的app带来伟大的API，但是你必须继续支持一些旧的Adroid版本直到其更新。
下面的仪表盘统计了当前设备的所有的Android版本的使用情况。
Platform Versions

如何支持各类不同的Android版本？
1、在AndroidManiest.xml文件中设置：

<manifest xmlns:android="http://schemas.android.com/apk/res/android" ... >
    <uses-sdk 
        android:minSdkVersion="4"         
        android:targetSdkVersion="15" />
        ...
</manifest>

2、在Runtime时检查系统的版本：

private void setUpActionBar() {
    // Make sure we're running on Honeycomb or higher to use ActionBar APIs
    if (Build.VERSION.SDK_INT >=     
            Build.VERSION_CODES.HONEYCOMB) {
        ActionBar actionBar = getActionBar();
        actionBar.setDisplayHomeAsUpEnabled(true);
    }
}

3、使用Android Support Libary：
Android Support Libary包含一些你的app可以使用的库。每一个库度支持指定范围的Android平台版本和一系列的特性。

通常推荐使用V4 support 和 V7 appcompat库。

   com.android.support:support-v4:23.3.0

com.android.support:appcompat-v7:23.3.0
